## 类型化数组

前置知识: js中的数字全是双精度浮点型  占64位
好处:节约内存空间

```js
类型化数组的创建方法
位数存储的范围大小
n位 无符号 0 ~ 2^n-1
n位 有符号 -2^(n-1) ~ 2^(n-1) - 1

var arr = new Int8Array() //每一个数组项占8位 也就是一个字节
new Int16rray() //每一个数组项占16位 也就是两个字节
new Int32rray() //每一个数组项占32位 也就是四个字节
arr.lenght //得到数组的长度
arr.byteLenght //得到数组的字节长度
```

注意点: 
- 类型化数组不能增加和删除元素  
- 一些返回数组的方法  返回的是同类型化的数组
